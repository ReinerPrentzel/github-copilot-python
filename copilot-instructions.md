# Copilot Instructions

## Project Overview
This project is a Flask-based web application for playing Sudoku. It includes:
- A Python backend (`starter/app.py`, `starter/sudoku_logic.py`) for puzzle generation and solution checking
- A frontend with HTML, CSS, and JavaScript for user interaction

## Coding Guidelines
- Follow PEP8 for Python code
- Use clear, descriptive variable and function names
- Keep frontend and backend logic separated
- Write modular, reusable functions
- Add comments for complex logic

## Copilot Usage
- Suggest Python code compatible with Flask and standard libraries
- For frontend, use vanilla JavaScript, HTML5, and CSS3
- Avoid using external dependencies unless specified in `requirements.txt`
- When generating Sudoku puzzles, ensure they have a unique solution

## Testing
- Prefer simple, self-contained test functions for logic in `sudoku_logic.py`
- Manual testing is sufficient for the web interface

## Documentation
- Keep `README.md` up to date with setup and usage instructions
- Add docstrings to all Python functions
